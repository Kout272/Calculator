ИНСТРУКЦИЯ ПО УСТАНОВКЕ БИБЛИОТЕКИ REQUESTS
===============================================

Для работы с актуальными курсами валют нужно установить библиотеку requests.

СПОСОБ 1 - Через командную строку:
1. Откройте командную строку (cmd) или PowerShell
2. Перейдите в папку с калькулятором: cd "c:\БГУИР_\calculaor"
3. Выполните команду: pip install requests

СПОСОБ 2 - Через Python:
1. Откройте командную строку
2. Выполните: python -m pip install requests

СПОСОБ 3 - Через скрипт:
1. Запустите файл install_requests.py: python install_requests.py

СПОСОБ 4 - Вручную:
1. Скачайте requests с https://pypi.org/project/requests/
2. Установите: pip install requests

ПРОВЕРКА УСТАНОВКИ:
===================
После установки запустите калькулятор. Если requests установлена,
конвертер валют будет использовать актуальные курсы с API Беларусбанка.

БЕЗ REQUESTS:
=============
Калькулятор будет работать с встроенными курсами валют (примерными).
Функциональность сохранится, но курсы могут быть неактуальными.

ПРОБЛЕМЫ С УСТАНОВКОЙ:
=====================
Если возникают проблемы с установкой:
1. Обновите pip: python -m pip install --upgrade pip
2. Попробуйте: pip install --user requests
3. Используйте встроенные курсы (калькулятор работает и без requests)
