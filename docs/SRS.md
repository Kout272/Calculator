# SRS — Multi‑Calculator

## Введение
Multi‑Calculator — десктопное приложение (Python/PyQt5) с вкладками: Калькулятор, График, Валюты, Кредит/Ипотека.

## Пользовательские требования
- UI: PyQt5, горячие клавиши, сообщения об ошибках.
- Визуализация: Matplotlib.

## Функциональные требования
- Калькулятор: безопасная оценка, история, AC/CE, '^'→'**'.
- График: ввод f(x), шаблон sin(x)/x при пустом поле, диапазоны, сохранение PNG.
- Валюты: выбор направлений, автообновление, кэш, офлайн.
- Кредит/Ипотека: аннуитет/дифф., график платежей.

## Нефункциональные
- Usability, Security (AST), Performance (<200мс на 200 символов), Compatibility (Win/macOS/Linux).

## Архитектура
Модули: main.py, calculator_ui.py, calculator_core.py, styles.py, utils.py.
