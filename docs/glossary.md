# Multi‑Calculator — Глоссарий

- **Expression** — вводимое математическое выражение.
- **Safe Eval** — безопасная оценка через AST с белыми списками.
- **History/last_result** — переиспользование последнего результата.
- **Plot** — построение функции f(x) на Canvas (Matplotlib).
- **Нормализация** — '^'→'**', запятая→точка.
- **Currency** — конвертация с курсами и кэшем/офлайном.
- **Finance** — аннуитет/дифференцированный платёж.
- **UI State** — Idle/Typing/Evaluating/ShowingResult.
